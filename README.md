# Flutter Animations Collection

A curated collection of 10 beautiful Flutter animations.

## Animations

1. **CRT Heart Animation** - Retro CRT monitor with animated hearts
2. **Luxury Knob Animation** - 3D knob with physics-based rotation
3. **Pitch Control Animation** - Rolling number display with blur effects
4. **Dot Eye Animation** - Animated eyes on dot matrix display
5. **Dot Music Visualizer** - Music visualization on dot matrix
6. **Dot Text Scroller** - Scrolling text on dot matrix
7. **Dot Weather Display** - Weather animations on dot matrix
8. **Dot Moon Phase Tracker** - Lunar cycle animation
9. **Pomodoro Timer** - Timer with dot matrix display
10. **Expandable Notification** - Smooth expand/collapse animation

## Usage

Each file is a standalone Flutter widget. Copy and import into your Flutter project.
