# Dot Matrix Animations

Collection of 64x64 dot matrix display animations.

## Included Animations

- **Eye Animation** - Animated eyes with pupil tracking
- **Music Visualizer** - Real-time music visualization bars
- **Text Scroller** - Horizontal scrolling text display
- **Weather Display** - Weather conditions with icons
- **Moon Phase Tracker** - 29-day lunar cycle visualization
- **Pomodoro Timer** - Countdown timer display
